## 🔥 프로젝트 소개

<mark>Knock는 사용자가 등기부등본을 업로드하면 파인튜닝된 AI모델을 통해 분석하고 사용자에게 결과를 제공해주는 서비스에요. 부동산 거래 시 숨겨진 위험 요소를 발견하고
일반인도 쉽게 이해할 수 있도록 위험도를 시각화하여 제공해요. 기존 등기부등본 해석에 소요되던 검색·문의 시간을 크게 줄이고 종합 점수와 함께 판단 근거를 제공하여 사용자는 전문적인 지식 없이도 등기부등본에 대한 전반적인 분석 내용을 보고 이해할 수 있어요.</mark>

## 🚀 설치 및 실행

### 개발 환경 설정

```bash
# 저장소 클론
git clone https://github.com/Hackathon-ROKANF/Knock-FE.git
cd Knock-FE

# 의존성 설치
yarn install

# 개발 서버 실행
yarn dev
```

### 빌드 및 배포

```bash
# 프로덕션 빌드
yarn build

# 빌드 결과 미리보기
yarn preview

# 린터 실행
yarn lint
```
